# ESP8266 Temperature & Humidity Data Logger

Αυτό το project χρησιμοποιεί έναν microcontroller ESP8266 (NodeMCU) και έναν αισθητήρα DHT22 για την περιοδική καταγραφή της θερμοκρασίας και της υγρασίας. Τα δεδομένα αποθηκεύονται τοπικά στη μνήμη flash της πλακέτας χρησιμοποιώντας το σύστημα αρχείων **LittleFS**.

## Βασικά Χαρακτηριστικά
* **Τοπική Καταγραφή:** Τα δεδομένα αποθηκεύονται σε ένα αρχείο `log.txt` μέσα στη μνήμη του ESP8266.
* **Συγχρονισμός Ώρας (NTP):** Συνδέεται στο WiFi αποκλειστικά για να λάβει την ακριβή ώρα και ημερομηνία μέσω NTP Server, ώστε κάθε καταγραφή (timestamp) να είναι σωστή.
* **Αυτόματη Διαχείριση Μνήμης:** Ο κώδικας ελέγχει συνεχώς το μέγεθος του αρχείου `log.txt`. Όταν το αρχείο ξεπεράσει τα 500KB, διαγράφεται αυτόματα και δημιουργείται νέο, αποτρέποντας το γέμισμα της μνήμης.
* **Offline Λειτουργία:** Αν χαθεί η σύνδεση στο WiFi, το σύστημα συνεχίζει να καταγράφει κανονικά (με τη διαφορά ότι τα timestamps μπορεί να μην είναι συγχρονισμένα).

## Απαιτήσεις Υλικού (Hardware) 🛠️
* NodeMCU v2 (ESP8266)
* Αισθητήρας θερμοκρασίας/υγρασίας DHT22
* Breadboard & Jumper wires (Καλώδια)
* Καλώδιο Micro-USB (για τροφοδοσία και προγραμματισμό)
